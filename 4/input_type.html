type="hidden" - 히든 필드


(name, value)

type="text" - 텍스트 필드


(name, size, value, maxlength)

type="password" - 비밀번호 입력란


(name, size, maxlength)

type="search" - 검색 상자


(value)

type="url" - URL 입력란


(http://로 시작, 공백x, name, size, value, maxlength)

type="email" - 메일 주소 입력란


(이메일 형식 자동 체크, name, size, value, maxlength)

type="tel" - 전화번호 입력란


(name, size, value, maxlength)

type="number" - 숫자 입력란


(텍스트 or 스핀 박스, min, max, value, step)

type="range" - 슬라이드 막대로 숫자 조절


(min, max, value, step)

type="radio" - 라디오 버튼(하나만 선택 가능)

option A
option B
(name, value-필수 속성(영or숫자), checked)

type="checkbox" - 체크박스(복수 선택 가능)

option A
option B
(name, value-필수 속성(영or숫자), checked)

type="color" - 색상 선택 상자


(지원 브라우저 제한, value-기본 색 설정)

type="date","month","week" - 날짜 표시

yyyy-mm-dd

yyyy-mm

yyyy-W24
(min, max, step, value, 브라우저마다 지원 형태가 다름)

type="time","datetime","datetime-local" - 시간 지정






(min, max, step, value)

type="submit","reset" - 서버 전송, 리셋 버튼

폼으로 정보 전송

재설정

type="image" - 이미지 버튼


(src="경로" alt="대체 텍스트", submit에 이미지 삽입 가능)

type="button" - 버튼


(value, onclick, on...)

type="file" - 파일 첨부


(버튼 형태는 브라우저에 따라 다름)
